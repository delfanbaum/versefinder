Versefinder

Verse Finder enables you to create poetry using texts from all around the web! To use Verse Finder:

  1. Go to any webpage you like, and use your cursor to select some text.
  2. Right click the selected text, and select Verse Finder:
    • To add the text to your poem and continue browsing, select Add Text.
    • To add the text to your poem and continue browsing, select Add Text and Go. This will open the poem maker page.
  3. When you're ready to start manipulating your found texts, click below to open the poem creator page and get to work!
